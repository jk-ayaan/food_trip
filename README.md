# 📍 부산에 가면 · When in Busan

부산관광포털 '부산에가면' **5개 섹션** + 우체국 추천 **우슐랭**, 총 **1,300여 곳**을 지도·다국어(한·영·일·중)로 둘러보는 모바일 반응형 웹앱.

> 데이터 출처: [부산관광포털 비짓부산](https://www.visitbusan.net/index.do?menuCd=DOM_000000201001000000) (부산광역시) · [우체국 추천 맛집가이드](https://github.com/jk-ayaan/usulleng) (부산지방우정청, 2024·2025)

## 섹션

| 섹션 | 수 | 비고 |
|---|---|---|
| 🏞️ 명소 | 212 | 종류(바다·전망·역사·박물관·테마·자연·거리) |
| 🍜 음식 | 337 | '부산의 맛' — 종류(고기·해산물·면·카페 …) |
| 📮 우슐랭 | 318 | 우체국 직원·집배원 추천 맛집 — **부산·울산·경남**, 추천 우체국·수록연도(24·25) 표시 |
| 🎉 축제 | 40 | 기간 + 진행중/예정/종료 상태 |
| 🏨 숙박 | 362 | 유형(호텔·게스트하우스·리조트) |
| 🛍️ 쇼핑 | 55 | 종류(백화점·전통시장·면세 …) |

## 기능

- 📱 **모바일 반응형** — 1열(모바일) → 4열(데스크톱), 상단 섹션 탭으로 즉시 전환
- 🗺 **지도 보기** — Leaflet + OpenStreetMap(CARTO), 핀·클러스터, 핀 클릭 시 정보
- 🌐 **다국어** — 한국어·English·日本語·中文 (비짓부산 공식 번역; 숙박·中文·우슐랭 일부는 한/영 표기)
- 🔎 **검색·필터** — 이름/지역/키워드 검색, 구·군 + 섹션별 종류, 정렬
- ✓ **가본 곳 체크** — 카드·지도 팝업에서 토글, 기기(localStorage)에 저장
- ♥ **찜 + 우선순위** — 찜한 곳에 높음/보통/낮음 우선순위 지정(⚑ 탭하면 순환), '내 저장' 필터·우선순위순 정렬
- 📞 전화 · 카카오맵 길찾기 · 비짓부산 상세 링크

데이터가 내장된 **단일 `index.html`** 로 서버 없이 열어도 동작합니다.

## 데이터 파이프라인

```bash
python3 grab.py all         # 5개 섹션 수집 + 좌표 + 영·일 번역 → data/<section>.json
python3 grab_usulleng.py    # 우슐랭(jk-ayaan/usulleng) → data/usulleng.json
python3 build_app.py        # data/*.json → index.html
```

| 파일 | 설명 |
|---|---|
| `grab.py` | 범용 수집기 — 섹션별 목록·상세 파싱, `ucg_seq` 구·군 매핑, 축제 기간 추출, 종류 자동 분류, 영·일 번역 |
| `grab_usulleng.py` | [usulleng](https://github.com/jk-ayaan/usulleng) `restaurants.json` → 본 앱 스키마 변환 |
| `build_app.py` | 다국어·지도·섹션탭 단일 HTML 생성기 |
| `data/*.json` | 섹션별 데이터셋(좌표·다국어 포함) |

## 주의

비짓부산·우체국 맛집가이드 공개 정보를 정리한 것으로 실제와 다를 수 있습니다. 방문 전 운영시간·휴무·행사기간을 확인하세요. 종류는 이름·메뉴·소개 기반 자동 분류입니다. 사진은 비짓부산 서버 이미지를 사용합니다. 가본 곳·찜 데이터는 브라우저 localStorage에만 저장됩니다.
