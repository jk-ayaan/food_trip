# 🍜 부산의 맛 · Taste of Busan

부산광역시 맛집 **337곳**을 지도와 다국어(한·영·일·중)로 둘러보는 모바일 반응형 웹페이지.

> 제공: **부산의 맛** (부산광역시, 2026) · 데이터 출처: [부산관광포털 비짓부산](https://www.visitbusan.net/index.do?menuCd=DOM_000000201002000000)

## 기능

- 📱 **모바일 반응형** — 1열(모바일) → 4열(데스크톱)
- 🗺 **지도 보기** — Leaflet + OpenStreetMap(CARTO), 337곳 핀·클러스터, 핀 클릭 시 가게 정보
- 🌐 **다국어** — 한국어·English·日本語·中文 (식당 정보는 비짓부산 공식 번역; 中文은 UI만 번역, 상세는 영문 표기)
- 🔎 **검색·필터** — 가게/메뉴/지역 검색, 16개 구·군 + 10개 음식 종류, 조회·좋아요·가나다 정렬
- 📞 전화 바로걸기 · 카카오맵 길찾기 · 비짓부산 상세 링크

페이지는 **단일 `index.html`** 로 데이터가 내장되어 있어 서버 없이 파일을 열어도 동작합니다.

## 데이터 파이프라인

```bash
python3 scrape.py 22   # 목록 22p → 상세 337곳 (이름·주소·전화·시간·메뉴·설명·구군)
python3 enrich.py      # 좌표(lat/lng) + 영·일 공식 번역 추가
python3 build.py       # data.json → index.html 생성
```

| 파일 | 설명 |
|---|---|
| `scrape.py` | 비짓부산 음식 목록·상세 수집, 구·군(`ucg_seq`) 매핑, 음식종류 키워드 분류 |
| `enrich.py` | 상세페이지 좌표 + `lang_cd=en/ja` 번역 수집 |
| `build.py` | 다국어·지도 포함 단일 HTML 생성 |
| `data.json` | 337곳 통합 데이터셋(좌표·다국어 포함) |

## 주의

데이터는 부산관광포털 공개 정보를 정리한 것으로 실제와 다를 수 있습니다. 방문 전 영업시간·휴무를 확인하세요. 음식 종류는 메뉴·소개 기반 자동 분류입니다. 사진은 비짓부산 서버 이미지를 사용합니다.
